# AGENTS.md

## Commit Policy

- When a change touches Lingo outputs, always commit Lingo cache and language files together in the same commit.
- Required include: `src/.lingo/cache/*` and related locale/language output files generated by Lingo.

## i18n Policy

- Do not hardcode user-visible language in code. Use the project's unified i18n flow.
- Treat free-form business data such as `banReason` as raw content. It does not need translation, but any surrounding UI copy must still use i18n.
